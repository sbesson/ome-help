CLS OMERO Server

CLS has an OMERO server, called Nightshade, that is available for use by all CLS staff and students to store, view and analyse image data.

An account will be automatically created for you when you log in for the first time.

This is easiest using the the Nightshade OMERO.web client in your usual browser:

www.nightshade.openmicroscopy.org

The desktop client, OMERO.insight can be downloaded from the OMERO Downloads page.

When you have created your account, you will be in a "default" group. To be able to share your and colleague's data, you need to join your Lab or PI's group. Ask the group owner, usually a PI to add you to their group or email the OMERO systems administrator and indicate which group(s) you would like to be in.

Note: As OMERO.insight is a stand-alone Java application, and does not require installation as such, you do not need administrator privileges to install it.

On Windows, expand the .ZIP archive and move the OMERO.clients folder to your Documents or home directory.

On the Mac expand the archive and drag it to the Applications folder.

Full "Getting Started" instructions specifically for using OMERO and Nightshade are on the CLS-Nighshade page.

The OMERO Help website contains detailed help documents for common workflows.

Using OMERO you can publish images stored on Nightshade directly on your website. See the Public Demo page for details.
